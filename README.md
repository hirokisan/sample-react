# SAMPLE-REACT

## VERSION
- npm 5.3.0
- node 8.5.0

## PRE SET UP
- install npm

## set up

```
export PATH=$PATH:./node_modules/.bin
npm install
```

## BUILD SERVER

```
npm start
```
